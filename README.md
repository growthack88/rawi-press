# Rawi Press

HTML/CSS/JS prototype of Rawi Press v2 — a newsroom tool with an Associated Press–inspired editorial aesthetic, RTL-first with Arabic as the default language.

Exported from [Claude Design](https://claude.ai/design) and published as a static site via GitHub Pages.

## Live preview

https://growthack88.github.io/rawi-press/

## Pages

- `index.html` — entry point
- `login.html`, `request-access.html`
- `dashboard.html`, `library.html`, `preview.html`
- `ai-studio.html`, `media-center.html`, `export.html`, `reports.html`

Shared tokens, scripts, and assets live in `assets/`.
